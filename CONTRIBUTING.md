# 🤝 Panduan Kontribusi - STG Project Monorepo

Terima kasih telah tertarik berkontribusi pada **Sovereign Titan Genesis (STG‑Project)**.  
Kontribusi komunitas adalah kunci agar proyek ini tumbuh dan berkembang.

---

## 📂 Struktur Kontribusi
- **planning/** → Tambahkan ide atau dokumen perencanaan proyek (jalan, jembatan, data center, e-commerce, dll).
- **execution/** → Tambahkan kode, laporan, atau hasil pengerjaan proyek.
- **docs/** → Tambahkan dokumentasi (audit, roadmap, brand guidelines).
- **scripts/** → Tambahkan utilitas, dashboard, atau automasi.
