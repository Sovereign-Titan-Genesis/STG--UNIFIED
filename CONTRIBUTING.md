# 🤝 Panduan Kontribusi - STG Project Monorepo

Terima kasih telah tertarik berkontribusi pada **Sovereign Titan Genesis (STG‑Project)**.  
Kontribusi komunitas adalah kunci agar proyek ini tumbuh dan berkembang.

---

## 📂 Struktur Kontribusi
- **planning/** → Tambahkan ide atau dokumen perencanaan proyek (jalan, jembatan, data center, e-commerce, dll).
- **execution/** → Tambahkan kode, laporan, atau hasil pengerjaan proyek.
- **docs/** → Tambahkan dokumentasi (audit, roadmap, brand guidelines).
- **scripts/** → Tambahkan utilitas, dashboard, atau automasi.

---

## 🚀 Cara Berkontribusi
1. **Fork repositori ini** ke akun GitHub Anda.
2. **Buat branch baru** untuk kontribusi Anda:


checkout -b feature/nama-kontribusi
3. **Tambahkan file atau edit** sesuai folder yang relevan.
4. **Commit perubahan** dengan pesan yang jelas:

 commit -m "Add: Perencanaan Data Center"
5. **Push branch** ke fork Anda:

push origin feature/nama-kontribusi
6. **Ajukan Pull Request (PR)** ke repositori utama.

---

## 🛡️ Aturan Kontribusi
- Gunakan bahasa yang jelas dan profesional.
- Sertakan dokumentasi singkat untuk setiap proyek baru.
- Pastikan kode atau dokumen Anda tidak melanggar hak cipta.
- Ikuti struktur folder yang sudah ditentukan.

---

## 🌐 Komunikasi
- Diskusi komunitas dilakukan melalui GitHub Issues.
- Untuk pertanyaan teknis, gunakan label **question**.
- Untuk bug, gunakan label **bug**.
- Untuk ide baru, gunakan label **enhancement**.

## 📜 Lisensi
